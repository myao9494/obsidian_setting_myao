# VSCode ガイド

## 1. マルチカーソル操作

### 基本的なマルチカーソル
- **`Cmd/Ctrl + D`**: 選択中の単語と同じ単語を次々に選択
- **`Cmd/Ctrl + Shift + L`**: 選択中の単語と同じ単語をすべて選択
- **`Option/Alt + Click`**: クリックした位置にカーソルを追加
- **`Cmd/Ctrl + Option/Alt + ↑/↓`**: 上下の行にカーソルを追加

### 活用例
複数の変数名を一括で変更したり、同じパターンの編集を複数箇所で同時に行う際に便利です。

## 2. コマンドパレット

- **`Cmd/Ctrl + Shift + P`**: コマンドパレットを開く
- **`Cmd/Ctrl + P`**: ファイル検索（Quick Open）
- **`Ctrl + R`: 最近開いたファイル・ワークスペースを開く

### よく使うコマンド
- `>Reload Window`: VSCodeをリロード
- `>Format Document`: ドキュメント全体をフォーマット
- `>Transform to Uppercase/Lowercase`: 大文字・小文字変換

## 3. 検索と置換

### 基本検索
- **`Cmd/Ctrl + F`**: ファイル内検索
- **`Cmd/Ctrl + Shift + F`**: プロジェクト全体検索
- **`Cmd/Ctrl + Shift + H`**: プロジェクト全体置換
